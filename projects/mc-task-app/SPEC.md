# 我的世界任务激励工具 — 设计规范

## 1. Concept & Vision

一个家长友好的儿童任务激励 App，用《我的世界》像素画风让孩子主动完成任务换取"开宝箱"的成就感。整体感觉是"一个真正的 Minecraft 小游戏"，而不是"教小朋友做任务"。视觉上极度还原游戏内像素方块、UI 字体和动画风格，让 3-10 岁孩子产生强烈的身份认同。

## 2. Design Language

**色彩：**
- 背景主色：#0a0a1e（深夜泥土感）+ AI生成 Minecraft 场景背景
- 主色调：#4CAF50（草地绿）
- 高亮/激活：#8BC34A
- 材质色：#9C6B30（木材）、#7D7D7D（石材）、#C0C0C0（铁）、#4DD0E1（钻石）
- 金色：#FFD700
- 文字主色：#FFFFFF
- 按钮色：#4CAF50（草地绿按钮）+ 线性渐变

**字体：**
- 标题：Press Start 2P（Google Fonts，像素字体）
- 正文：ZCOOL KuaiLe（Google Fonts，手写风格）

**背景：**
- 每个 Tab 页面对应专属 Minecraft 场景背景（AI 生成）
  - 任务页：草地平原日出风景
  - 宝箱页：地下宝箱房间
  - 收藏页：合成台物品陈列
  - 爸妈页：主世界全景
- 所有背景覆盖半透明遮罩，保证文字可读性
- 毛玻璃（backdrop-filter: blur）应用于所有卡片和面板

**动效：**
- 卡片进入：fadeSlideUp 0.25s ease
- 悬浮粒子：任务页背景有浮动 Emoji 装饰
- 宝箱悬浮：float 动画
- 开箱：方块弹出 bounce 动画
- 套装解锁：金光芒动画
- Tab 切换：背景图 crossfade 0.6s ease

**动效哲学：**
- 开箱：盖子弹起 + 方块弹出弹跳动画（spring bounce），500ms
- 获得道具：星星粒子 + 放大淡入
- 套装合成：金光汇聚 + 套装图标掉落动画
